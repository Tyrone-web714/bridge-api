# Asset Provenance Register

Runtime branding is neutral. The current 32-direction white truck marker was
supplied as project input and contains no visible third-party logo.

Before external commercial distribution, the asset owner must record the source
files, creator, creation date, and written commercial-use rights. Until that
record exists, treat the truck sprites as pilot-only assets. Do not reintroduce
customer logos, branded vehicles, or product trade dress.
