# Authorization and Permissions

New permissions:

- `fleet_score.view`
- `fleet_score.manage`
- `fleet_score.calculate`
- `fleet_score.benchmark`
- `platform.fleet_score.support`

Default access:

- Platform Admin: all FISS permissions.
- Organization Admin: manage, calculate, view, and private benchmark permissions.
- Supervisor: view, calculate, and private benchmark permissions.
- Driver: view only where surfaced.
- Warehouse Employee: view only where surfaced.

FISS follows private-by-default API security.
