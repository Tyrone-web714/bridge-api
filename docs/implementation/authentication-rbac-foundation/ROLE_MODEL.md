# Role Model

Approved code constants:

- `PLATFORM_ADMIN`
- `ORGANIZATION_ADMIN`
- `SUPERVISOR`
- `DRIVER`
- `WAREHOUSE_EMPLOYEE`

Unsupported business labels such as Dispatcher, Auditor, Safety Manager, Depot Manager, Maintenance, Customer Service, and Executive Viewer are not roles. They must be modeled as responsibilities or permission sets assigned under one of the approved roles.
