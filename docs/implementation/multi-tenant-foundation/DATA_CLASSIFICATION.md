# Data Classification

This phase classifies data into the governance-approved categories.

## Platform-Global

- Low-clearance bridge reference data
- Truck restriction reference data
- Platform configuration explicitly documented as global
- Approved future shared safety intelligence after moderation

## Organization-Private

- Admin users
- Drivers
- Driver sessions
- Warehouse employees
- Customers
- Routes and manifests
- Stops
- Delivery notes
- Delivery records
- Route inventory
- Receipts and document metadata
- Account intelligence and Organization-specific analytics
- Supervisor alerts and reports

## Shared-Approved Safety

Not implemented in this phase. Driver-submitted or Organization-submitted hazards remain Organization-private until a future Platform Admin approval and sanitization workflow exists.

## Derived Or Temporary

Derived analytics, temporary queues, and generated reports are Organization-private unless explicitly documented as platform-global reference data.
