# Shared Safety Moderation UI

This phase adds a server-rendered Platform Admin moderation interface for Shared Safety Intelligence.

Scope:

- moderation queue
- candidate detail view
- sanitization form
- approve, reject, request correction, duplicate, merge/link, retire, and supersede actions
- audit trail visibility
- Platform Admin-only navigation and direct URL protection

Out of scope:

- AI moderation
- automatic scoring
- reputation systems
- BI/KPI dashboards
- mobile redesign
