# Data Subject Requests

`data_subject_requests` supports `ACCESS`, `EXPORT`, `CORRECTION`, `DELETION`, and `RESTRICTION`.

Supported decisions are `DELETE`, `ANONYMIZE`, `PARTIALLY_DELETE`, `RETAIN_UNDER_POLICY`, `RETAIN_UNDER_LEGAL_HOLD`, and `REJECT_WITH_DOCUMENTED_REASON`.

The workflow is jurisdiction-neutral and does not claim legal compliance beyond reviewed scope.
