# Drill-Down and Lineage

Every KPI snapshot preserves enough metadata to answer:

- Which formula version was used?
- Which raw inputs were provided?
- Which time period was calculated?
- Which subject was calculated?
- When was it calculated?
- Who initiated calculation?
- Was the result good, warning, critical, stale, missing, or failed?

Drill-down endpoints are Organization-scoped and cannot read another Organization's snapshots.
