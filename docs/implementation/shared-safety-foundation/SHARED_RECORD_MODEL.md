# Shared Record Model

`shared_safety_records` contains approved platform-global safety facts.

Shared read responses include:

- `id`
- `hazardType`
- `latitude`
- `longitude`
- `geometry`
- `severity`
- `verificationStatus`
- `effectiveFrom`
- `effectiveTo`
- `status`
- `sourceClassification`
- `confidence`
- `evidenceLevel`
- `approvedBy`
- `approvedAt`
- `sanitizedMedia`

Ordinary shared reads do not expose source Organization identifiers or private source evidence.
