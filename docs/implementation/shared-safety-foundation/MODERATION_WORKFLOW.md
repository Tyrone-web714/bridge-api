# Moderation Workflow

Lifecycle:

1. Driver or authorized Organization user submits a hazard.
2. The hazard remains Organization-private.
3. An authorized Organization reviewer may nominate it for platform review.
4. Platform Admin reviews the moderation candidate.
5. Platform Admin sanitizes the candidate.
6. Platform Admin may approve, reject, mark duplicate, or retire related shared records.
7. Only approved sanitized records become platform-global.

No automatic publication exists in this phase.
