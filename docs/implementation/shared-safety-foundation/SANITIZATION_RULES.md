# Sanitization Rules

Approval requires sanitized content.

Sanitized shared records must exclude:

- Organization names
- driver identity
- customer names or account numbers
- delivery details
- route or manifest numbers
- receipts
- KPI or performance data
- internal notes
- private media URLs or metadata

Allowed content:

- hazard type
- sanitized description
- sanitized coordinates or geometry
- severity
- verification and evidence classification
- approved sanitized media references only

If a candidate cannot be safely sanitized, it must be rejected or remain private.
