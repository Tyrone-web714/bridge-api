# Remaining Media Security Work

## Open Items

- Migration or safe compatibility transition for the 3 verified production delivery-note media items with direct public R2 current URLs.
- Backfill plan for legacy public media URLs.
- Decision on when to disable R2 public access.
- Dedicated media table or lifecycle object-reference integration, if required by ODR-019 implementation maturity.
- Physical mobile and authenticated dashboard media-view regression after deployment.
- Hazard-report media route expansion if production hazard media requires the same authenticated media endpoint.

## Non-Goals For This Phase

This phase intentionally does not mutate production media, alter R2 bucket access, or create new storage lifecycle policies.
