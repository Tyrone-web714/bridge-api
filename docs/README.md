# Truck-Safe Routing Documentation

This documentation governs the complete Truck-Safe Routing platform.

Truck-Safe Routing is currently split across multiple repositories and local development folders. The backend API, supervisor dashboard, routing services, database integration, hazard data, and Render deployment configuration are located in this GitHub repository.

The active mobile application currently exists locally at:

`C:\dev\tsr-mobile`

The mobile application does not yet have a configured GitHub remote. The mobile source code is not contained in this repository at this time.

The documentation in this repository applies to the full Truck-Safe Routing platform, not only the backend. Future migration to a monorepo or to a coordinated multi-repository structure requires a separate approved plan.

Before architectural work, Codex must read:

`docs/architecture/CODEX_START_HERE.md`
